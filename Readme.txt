# Tahiti Time Numerique

Une application web permettant de gérer et visualiser différents fuseaux horaires.

## Fonctionnalités

- Recherche et ajout de fuseaux horaires
- Affichage en temps réel de l'heure dans différents fuseaux
- Modification et suppression des fuseaux horaires
- Convertisseur de temps entre les fuseaux horaires

## Prérequis

Pour lancer l'application, vous devez avoir installé :
- Docker Desktop
- Docker Compose

## Lancer l'application

L'application utilise Docker pour faciliter le déploiement :
docker-compose up --build

## Accédez à l'application

Frontend : http://localhost:5173
API Backend : http://localhost:8080

## Architecture technique

Backend : Java 17 avec Spring Boot
Frontend : React avec Vite
Base de données : H2 (en mémoire)
